Программа распознавания положения и значения тримино на темносинем фоне.

Для запуска необходим python 3.7

А так же библиотеки:
 - scipy
 - numpy
 - sklearn (scikit-learn)
 - Pillow
 - matplotlib

ВАЖНО: Папки нельзя переименовывать или удалять оттуда находящиеся там файлы, иначе
программа работать не будет!

Файлы для обработки программой помещайте в папку "данные". Для удобства тестирования 
можно использовать Jupyter-notebook, в котором уже  есть примеры тестирования программы 
на предоставленных изображениях. Основная программа называется detector.py. После запуска 
вам будет предложено ввести имя файла в папке "данные". Далее будет выполнен реализованный
алгоритм, а его результат выведен в папку output. Выход программы представляет собой
текстовый файл "predictions.txt",  а так же два изображения, визуализирующие положение
тримино и их центров.

Время выполнения программы примерно 3-4 минуты.
